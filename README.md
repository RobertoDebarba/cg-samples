## How to build

### Prerequisites

* JDK 8 **32 bits**
* Maven
* Unzip tool
* Linux based O.S.

### Build and run

1. Extract libs.zip in project root dir
1. Import project on your IDE as a maven project
1. Configure Native Library Locations for gluegen lib (for each module): add all .so files from libs folder.
